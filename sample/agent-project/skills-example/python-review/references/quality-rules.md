# 代码质量规则

## 规模限制
- 单函数不超过 50 行
- 单文件不超过 500 行
- 单类不超过 20 个方法

## Import 规范
- 按顺序分组：标准库 → 第三方 → 本地模块，组间空一行
- 禁止 `from module import *`
- 禁止循环导入

## 异常处理
- 捕获具体异常，不用裸 `except:`
- 不要吞掉异常：`except Exception as e: pass` 是反模式
- try 块尽量小，只包裹可能出错的代码

## 可读性
- 魔法数字要命名：`if x > 3` → `MAX_COUNT = 3; if x > MAX_COUNT`
- 过长参数列表（>4个）用数据类封装
- 嵌套超过 3 层考虑提前返回
